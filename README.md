## Description
This repository contains the data analysis of 5-hydroxy-methyl-cytosine (5hmC) expression in 55 cases diagnosed as urothelial carcinoma of urinary bladder. The files associated with this analysis are the following:

* [Descriptive analysis](https://github.com/alcideschaux/5hmC-Bladder/blob/master/01_B5.md)
* [Expression in tumor and nontumor tissue](https://github.com/alcideschaux/5hmC-Bladder/blob/master/02_B5.md)
* [Expression and outcome](https://github.com/alcideschaux/5hmC-Bladder/blob/master/03_B5.md)
* [Expression and survival](https://github.com/alcideschaux/5hmC-Bladder/blob/master/04_B5.md)
* [Expression and regression models](https://github.com/alcideschaux/5hmC-Bladder/blob/master/05_B5.md)
